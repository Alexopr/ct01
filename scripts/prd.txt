# Product Requirements Document

## Проект: Java CRUD с админкой

### Первая стадия: Базовый CRUD + админка

- Реализовать backend на Java (Spring Boot 3.x, Java 17)
- Использовать Maven, JPA, PostgreSQL
- CRUD для основной сущности (например, User)
- Реализовать админ-панель (Spring Security, роли: admin/user)
- REST API с валидацией, логированием, тестами (JUnit 5)
- Документация через Springdoc OpenAPI
- Docker для запуска проекта
- Redis для кэширования

### Вторая стадия: Расширение и фронтенд

- Добавить новые инструменты (расширяемый модульный подход)
- Реализовать современный стильный frontend (React) 
- Интеграция frontend с backend через REST API
- Улучшить UX/UI, добавить авторизацию на фронте
- Поддержка мобильных устройств
